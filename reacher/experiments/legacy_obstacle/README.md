# Legacy Reacher Obstacle Experiments

This package preserves reusable source that previously lived beside generated
artifacts under `reacher/Haoran_obs_data/`.

The model checkpoints, datasets, and start/goal pairs remain in
`reacher/Haoran_obs_data/`. The collector still depends on the historical
`reacher.plan.obs_data_collect` module, which is not currently present in this
repository.
